UI Testing Playground Automation using Cypress

Automated UI testing project for [UI Testing Playground](https://uitestingplayground.com) built with **Cypress**.  
Covers multiple interactive pages with positive and negative test cases to validate UI functionality and stability.


** Pages Covered**
Dynamic ID · Class Attribute · Click · Hidden Layers · Load Delay · AJAX Data · Mouse Over · Scrollbars · Text Input · Verify Text · Sample App · Non-Breaking Space
